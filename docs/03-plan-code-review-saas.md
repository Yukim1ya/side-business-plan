# プランB: AIコードレビュー SaaS

## 概要

GitHub の Pull Request に対して Claude API が自動でコードレビューコメントを付加する **SaaS サービス**。月額課金モデルで継続収益を得る。

---

## サービス概要

```
開発者が PR を作成
       ↓
GitHub Webhook が発火
       ↓
Claude API がコードを解析
  - バグ・セキュリティ脆弱性の指摘
  - コーディング規約の確認
  - パフォーマンス改善提案
  - テスト不足の指摘
       ↓
PR に自動でレビューコメントが付加される
```

---

## 料金プラン

| プラン | 対象 | 月額 | PR数上限 |
|--------|------|------|---------|
| 個人 | 個人開発者 | 500円 | 30PR/月 |
| チーム | 5人以下のチーム | 2,500円 | 200PR/月 |
| スタートアップ | 20人以下 | 8,000円 | 無制限 |
| エンタープライズ | 要問い合わせ | 要相談 | 無制限+カスタム |

---

## 技術スタック

| コンポーネント | 技術 | 理由 |
|-------------|------|------|
| API サーバー | Python (FastAPI) | 既存スキルと一致 |
| GitHub連携 | GitHub Apps API | Webhook受信・コメント投稿 |
| AI エンジン | Claude API (Sonnet) | コードレビュー精度が高い |
| 課金 | Stripe | 月額サブスク対応 |
| インフラ | Railway / Render | 低コスト・簡単デプロイ |
| DB | Supabase (PostgreSQL) | 無料枠あり |

## 競合との差別化ポイント

| 競合 | 弱点 | 本サービスの優位性 |
|------|------|-----------------|
| GitHub Copilot | 英語中心・高価 | 日本語対応・低価格 |
| CodeRabbit | 海外サービス | 日本語UI・日本語コメント |
| 自前実装 | 開発コスト高 | すぐ使える・保守不要 |

## リスク

- 競合が多い市場
