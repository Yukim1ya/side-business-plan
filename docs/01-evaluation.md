# プラン評価基準・比較マトリクス

## Claude Code が副業に向いている理由

Claude Code（Claude API）は以下の作業を高精度で自動化できる:

| 自動化できる作業 | 副業への応用 |
|---------------|------------|
| 文書・レポート生成 | コンサルティング成果物、ブログ記事 |
| コード生成・レビュー | 開発支援サービス、SaaS |
| データ分析・解釈 | 調査レポート、競合分析 |
| テンプレートへの情報埋め込み | 提案書・仕様書の量産 |
| 多言語変換 | 翻訳・ローカライズサービス |

**ポイント**: Claude Code は「人間の判断 + AI の作業速度」を組み合わせることで、**単価を下げずに処理件数を増やす**ことが可能。

## 評価基準

| 軸 | 説明 |
|----|------|
| **月収ポテンシャル** | 安定稼働時の月収見込み |
| **開始難易度** | 副業開始までの障壁（技術・資格・初期費用） |
| **Claude Code 活用度** | どれだけ作業を自動化できるか |
| **スキル活用度** | 情報処理安全確保支援士・セキュリティ知識の活用度 |
| **スケーラビリティ** | 受注が増えたときに収益が伸びるか |
| **法的リスク** | 副業規制・資格法・税務リスク |

## 開始フェーズ推奨順序

```
Phase 1（今すぐ）: プランA スタート
  → 資格・知識がそのまま武器になる
  → Claude Code で成果物生成を自動化
  → クラウドソーシングで1件目の受注を目指す

Phase 2（3〜6ヶ月後）: プランC 並行開始
  → プランAの実績・ノウハウをブログ記事化
  → SEO流入で問い合わせを増やす

Phase 3（6〜12ヶ月後）: プランB 開発
  → プランA・Cで得た資金・知名度をもとにSaaS開発
  → 継続的な月額収入の柱を作る
```
