# プランC: 技術ブログ自動化 × マネタイズ

## 概要

専門知識（セキュリティ・Splunk・AD攻撃・Nutanix等）を持つニッチ領域で、Claude Code を使って技術記事を半自動生成し、複数の収益源から安定した副収入を得る。

---

## 狙うニッチ領域

競合が少なく専門性が評価されるテーマに絞る:

| テーマ | 競合 | 収益性 | 推奨度 |
|--------|------|--------|--------|
| Splunk SPL・検知ルール | 少ない | 高（企業需要） | ★★★★★ |
| AD攻撃検知（Blue Team） | 中程度 | 高 | ★★★★☆ |
| 情報処理安全確保支援士 対策 | 多い | 中 | ★★★☆☆ |
| Nutanix CE 構築 | 少ない | 中 | ★★★★☆ |
| 技術士（情報工学）対策 | 少ない | 中 | ★★★★☆ |

---

## Claude Code による記事生成フロー

```
① テーマ・キーワード選定（手動・30分）
       ↓
② Claude Code で構成案を生成
   - 見出し構成
   - 各セクションのドラフト
   - コードサンプル生成
       ↓
③ 実体験・実機検証結果を加筆（手動・1〜2時間）
   ← ここが差別化ポイント（実際に試した内容）
       ↓
④ 図・スクリーンショットを追加
       ↓
⑤ note / Zenn / 自社ブログ に投稿
```

**記事1本あたりの作業時間**: 従来4〜6時間 → **Claude Code活用後 2〜3時間**

## 収益モデル

### note 有料記事

| 記事種類 | 価格 | 月20本投稿時の想定月収 |
|---------|------|---------------------|
| Splunk 検知ルール集 | 500〜1,000円 | 記事ごとに5〜20購入 |
| AD攻撃シナリオ解説 | 800〜1,500円 | 専門家需要あり |
| 試験対策まとめ | 300〜500円 | 広く浅く販売 |

### Zenn（技術書典・スポンサー）

- 無料記事でSEO集客 → 有料本・サポートへ誘導
- 技術同人誌（PDF）を1,000〜3,000円で販売

### Google AdSense / アフィリエイト

- Udemy コースのアフィリエイト
- Amazon（技術書）アフィリエイト
- セキュリティツール（VPN・ウイルス対策）のアフィリエイト
