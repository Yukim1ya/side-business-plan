## 競合空白地帯（差別化できるポジション）

```
                    教育性（高）
                        ↑
         B2B研修ツール  |  ← 本企画が狙うゾーン
         （高価・法人向け）|  「ITエンジニア向け・日本語・
                        |   Steamで個人が買える教育ゲーム」
─────────────────────────┼─────────────────────────────→ ゲーム性（高）
                        |
   Project Manager (Initiation)  Project Manager SIM
   （地味・欧米向け）           （欧米カジュアル向け）
                        ↓
                    教育性（低）
```

**空白地帯の定義**:
- **日本語** に特化したIT PM シム → Steam日本語レビューでの発見性
- **エンジニアが共感できる「あるある」** → ゲーマーでなくIT層へのリーチ
- **B2Cで買えるが教育品質** → 研修ツールへの入口にもなる

## 戦略的示唆

### やるべきこと（差別化の核）

1. **日本語・日本のIT文化特化**: 「デスマーチ」「仕様変更地獄」「客先常駐」など日本特有のあるあるをゲームに入れる → 欧米競合が真似できない
2. **エンジニアコミュニティへの発信**: Qiita・Zenn・X のITクラスタへ「エンジニアが作ったPMゲーム」として売り込む
3. **B2B研修への橋渡し**: B2Cで認知 → 企業研修担当者へのアプローチへ

### 注意すること

1. **Project Manager SIM のリリースを常にウォッチ**: 先に大きな話題になるとポジションを取られる
2. **Game Dev Tycoonの亜種にならない**: 「ゲーム会社でなくITプロジェクト」という違いを明確に打ち出す
3. **早期アクセスの活用**: IT Specialist Simulator は早期アクセスで358件のフィードバックを得た → 同じ戦法が有効
