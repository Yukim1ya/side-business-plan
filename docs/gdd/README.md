## 設計サマリー

**コアパラメータ**: 予算 / 残り日数 / 進捗 / 品質スコア / チームモラル / クライアント満足度 / 技術的負債

**イベントカード20枚の内訳**:
- 🐛 技術系: 5枚（バグ・脆弱性・技術的負債・テスト自動化・サーバーダウン）
- 👔 クライアント系: 5枚（仕様変更・役員乱入・競合リリース・追加予算・直前修正依頼）
- 👥 チーム系: 5枚（退職・新メンバー・コミュニケーション断絶・燃え尽き・内部対立）
- 🌐 外部系: 3枚（クラウド障害・法改正・競合OSS公開）
- ✨ ポジティブ系: 2枚（自発改善提案・マイルストーン達成）
