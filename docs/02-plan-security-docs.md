## Claude Code による自動化フロー

```
① クライアントがヒアリングフォームに回答
   （会社規模・業種・既存ポリシー有無・懸念事項など）
         ↓
② Claude Code がヒアリング内容を解析
   → 業種・規模に合ったテンプレートを選択
   → 空白部分に会社固有の情報を埋め込み
   → 一貫性チェック・用語統一
         ↓
③ ドラフト自動生成（所要時間: 10〜30分）
         ↓
④ あなたが専門家として確認・加筆修正（1〜2時間）
         ↓
⑤ 最終版を納品（Word / PDF / Markdown）
```

**従来の手作業**: 1件 20〜40時間  
**Claude Code 活用後**: 1件 2〜4時間（**90%削減**）

## Claude Code 実装例

```python
# ヒアリング内容からセキュリティポリシーを生成する例
import anthropic

def generate_security_policy(hearing_data: dict) -> str:
    client = anthropic.Anthropic()
    
    prompt = f"""
    以下のヒアリング情報をもとに、情報セキュリティポリシーを作成してください。
    
    会社情報:
    - 業種: {hearing_data['industry']}
    - 従業員数: {hearing_data['employees']}
    - 主要なシステム: {hearing_data['systems']}
    - 懸念事項: {hearing_data['concerns']}
    
    要件:
    - ISMS（ISO27001）の考え方に準拠
    - 中小企業が実際に運用できる現実的な内容
    - 法令（個人情報保護法・不正アクセス禁止法）への言及を含む
    """
    
    message = client.messages.create(
        model="claude-opus-4-7",
        max_tokens=8096,
        messages=[{"role": "user", "content": prompt}]
    )
    return message.content[0].text
```
